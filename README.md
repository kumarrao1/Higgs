# CDXWriter
Download the FIRSTDRAFT.zip folder and unzip it.
Include your files in the relevant folders - Source, Reference, Guidelines, Templates.  Retain existing files in Guidelines and Templates and add new files to them.
Install VSCode and Codex by using prompts provided in the folder - readme.md
Install dependent files (pandoc, textutil etc) as listed in the readme.md
Open VS Code and open the FIRSTDRAFT workspace.
Converse with codex  - 'Create first draft'   or 'Review the <doc> in Source folder' and get your updated docs in the Output folder.
