# CDXWriter
1. Download the FIRSTDRAFT.zip folder and unzip it.
2. Include your files in the relevant folders - Source, Reference, Guidelines, Templates.  Retain existing files in Guidelines and Templates and add new files to them.
3. Install VSCode and Codex by using prompts provided in the folder - readme.md
4. Install dependent files (pandoc, textutil etc) as listed in the readme.md
5. Open VS Code and open the FIRSTDRAFT workspace.
6. Converse with codex  - 'Create first draft'   or 'Review the <doc> in Source folder' and get your updated docs in the Output folder.
